# Nikita Singh — Portfolio

Personal developer portfolio built with React.js.

**Live:** _(add your deployed URL here)_

## Tech Stack

- React.js 18
- Plain CSS with CSS custom properties (no UI library dependency)
- Google Fonts: Inter + Geist Mono

## Run Locally

```bash
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
```

## Structure

```
src/
├── components/     # Navbar, Hero, About, Skills, Experience, Projects, Contact, Footer
├── data/           # portfolio.js — all content in one place
└── styles/         # global.css — design tokens + shared styles
```

## Updating Content

All content (experience, projects, skills, personal info) lives in `src/data/portfolio.js`.
Edit that file to update anything — no need to touch component code.
